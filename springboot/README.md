### 使用感受

* spring的核心是aop和依赖注入, 围绕着这个核心提供了很多扩展点和方便程序开发的工具.

* springboot简化了spring的依赖管理,  提供了自动配置的功能

* springmvc 提供了基于servlet的web开发工具
